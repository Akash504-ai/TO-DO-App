# 📝 TO-DO App

A simple React-based To-Do application to manage daily tasks.  
Built using **React Hooks**, functional components, and styled with **custom CSS**.

## ✅ Features

- Add tasks ✅  
- Delete tasks ❌  
- Mark tasks as done ☑️  
- Stores tasks in **LocalStorage** 💾  
- Clean and modern UI 🎨
