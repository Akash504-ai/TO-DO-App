# TO-DO App 📝
A simple React-based To-Do application to manage daily tasks.
Built with React Hooks, functional components, and styled with CSS.

## Features
- Add tasks ✅
- Delete tasks ❌
- Mark tasks as done ☑️
- Stored in LocalStorage 💾

## How to run
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
